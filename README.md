# 为什么会这个仓库？
相信很多人看到炼丹蓝图组织下有front和frontend两个仓库，但是这两个仓库有什么区别呢？
- front是react开发的仓库
- frontend是一个vue仓库
这两个仓库就是负责不同方向的开发，一个是react，一个是vue。
front仓库的核心功能已经完成，frontend这个仓库还在进行开发。用了更好的架构设计，因为我个人喜欢vue的组合式api，react那种太难受了qwq。


## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
